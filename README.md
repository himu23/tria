# Contact List App (React)

Overview  
A simple contact-list web application built in React. Users can:
- View contacts
- Search contacts by name
- (Optional) Add a new contact

Tech Stack  
- React
- CSS / Tailwind / Material UI
- Mock API or hardcoded data

Demo Link  
https://tria-mu.vercel.app/

Features

View list of contacts
Search contacts by name
Add new contact

How to Run Locally

git clone https://github.com/himu23/tria.git
cd contact-app
npm install
npm start
